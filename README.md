# Retail Sales SQL Project

## Project Overview
Designed and implemented a relational database for a retail store using SQL. The project includes database creation, table relationships, and business-oriented sales analysis queries.

## Database Structure
- Customers
- Orders
- Products
- Order_Details

Primary and foreign key relationships were implemented to maintain data integrity.

## Key Analysis Performed
- Total revenue calculation
- Category-wise sales performance
- Top-spending customers identification
- Monthly revenue trends
- Multi-table JOIN operations

## Tools Used
- MySQL
- SQL (JOIN, GROUP BY, Aggregate Functions)

## Business Insight
The analysis helped identify repeat customer contribution and revenue trends across categories and time periods.
